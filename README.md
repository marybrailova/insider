# Инсайдер

## Описание проекта

**Web-приложение «Инсайдер»** — это игра-симулятор фондового рынка, в которой пользователь выступает в роли инвестора и принимает решения в условиях неполной информации. В приложении моделируется рынок с тремя компаниями, для которых каждый игровой день генерируются новости и слухи, а также доступна покупка инсайда — достоверной информации о завтрашнем изменении фундаментальной ценности компании

Пользователь может покупать, продавать или удерживать акции, стремясь максимизировать свой капитал за фиксированное число игровых дней. Использование инсайдов связано с риском проверки со стороны регулятора и возможными штрафами, что заставляет учитывать не только потенциальную прибыль, но и риск наказания.

Приложение демонстрирует ключевые экономические механизмы: информационную асимметрию, влияние ожиданий и поведения участников на рыночные цены, формирование рыночных пузырей и стратегию принятия решений в условиях неопределённости. Проект предназначен для наглядного изучения принципов работы финансовых рынков в игровой форме.

## Цель проекта

Создание интерактивного симулятора, позволяющего пользователю:
- понять влияние информации на динамику цен активов
- изучить эффект информационной асимметрии
- проанализировать риск и последствия инсайдерской торговли
- развить навыки стратегического принятия решений

## Основной функционал

- симуляция фондового рынка с несколькими компаниями
- ежедневная генерация новостей и слухов
- возможность покупки инсайда с риском проверки
- торговля акциями (покупка, продажа, удержание)
- подсчёт капитала и определение результата по итогам игры
